Open iHerb app on mobile phone
Tap on "Human" icon (right bottom corner)
Tap on "Sign In" link
Tap on "Create an Account" link
Enter a valid mobile number
Enter an invalid email address (without the "@")
Enter a valid password
Confirm the valid password
Check the "I represent..." checkbox
Tap on "CREATE AN ACCOUNT" button
Verify if an error message appears ("Please include an '@' in the email address. 'dddaddd.com' is missing an '@'.")
